# Feed Watcher GraphQL

## GraphQL (PORT 3002)

### Stack

* TypeScript
* Apollo Server
* Nexus
* GraphQL

### Installation

Make sure you have the `.env` file in the root of the folder with the `API_KEY` set.

```
yarn install
yarn run dev
```

### Heroku

This app is live at Heroku (`http://feed-watcher-graphql.herokuapp.com`)

### Running tests

```
yarn test
```
